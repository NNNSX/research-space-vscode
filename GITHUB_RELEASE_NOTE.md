# Research Space v2.1.1-alpha.1

`v2.1.1-alpha.1` 是基于上一 GitHub 发布版 `v2.1.0-alpha.41` 汇总后的新一轮 alpha 更新。

> 这是 **alpha 测试版**，稳定性弱于 `v1.2.2`。建议先备份工作区，再在副本中试用。

## 新增

- 蓝图容器进一步收口为更稳定的 overlay 外壳
- 蓝图容器补充运行历史、输出历史、失败后继续等运行态入口
- 对“终点 function 没有显式输出节点”的蓝图，自动补齐隐式 output slot
- 输入 / 输出占位开始真正共用同一套 Placeholder Shell

## 修复与优化

- 旧蓝图实例、旧画布结构、旧节点组残留开始进入更严格的迁移与清理
- 蓝图最终输出的持久化 / 恢复链路重做，重开画布后输出掉线问题持续收口
- 真实输出节点固定落到蓝图外侧，减少重开后的跳变和错位
- 输出拓扑统一为 `function → output placeholder → real output`
- 蓝图 direct pipeline 生成的中间结果改为按隐藏内部节点处理，不再弹到蓝图外
- 输入 / 输出占位内容开始按实际信息自动扩高，修复多行提示被遮盖问题
- 回收占位测高导致的边界异常增长问题
- 删除输入占位 footer 中与主体重复的提示、chips 和定位按钮
- 修复蓝图创建弹窗中拖拽选择文字会误触关闭的问题
- 修复普通文件节点 footer 被内容区挤出边界的共享布局回归
- Copilot 默认模型改为 `gpt-4.1`，并移除“自动”模型选项
