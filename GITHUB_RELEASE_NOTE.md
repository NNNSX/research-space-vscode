这是一次面向 AI 输出来源追溯与写作交付体验的 alpha 更新。

## 新增

- AI 文本输出会记录 `[资料1]`、`[资料2]` 等正文内引用的覆盖情况。
- AI 输出节点底部新增引用覆盖状态，可快速看到已引用来源数量。
- 引用状态支持悬浮查看详情，包含已引用来源、未引用来源、未连接标签和引用提醒。
- 选中节点导出 Markdown 时，会带出引用覆盖、缺失来源和来源节点列表。

## 修复与优化

- 文本工具提示词进一步强调多来源综合引用、不要伪造来源、不要只在末尾列来源。
- 引用缺失提示从“是否存在任意引用”升级为“每个连接来源是否被实际引用”。
- 引用展示逻辑独立为可测试模块，减少后续 UI 调整时破坏来源追溯信息的风险。

## 注意事项

- 建议在试用 alpha 版本前备份工作区。
- AI 引用覆盖检测只检查 `[资料1]` 这类标签是否出现，不等同于自动核验引用内容真实性；正式写作前仍建议人工复核。
- API Key 只应配置在本机 VS Code 设置中，不应写入项目文件。
