这是 `v2.1.1-alpha` 系列自上一 GitHub Release `v2.1.1-alpha.1` 以来的一次汇总更新，重点收口蓝图恢复稳定性、宠物小游戏体验和关键交互细节。

## 新增

- 宠物空间新增 **2048、数独、像素鸟** 小游戏，并保留贪吃蛇入口
- 小游戏面板开始长期记录 **最近分数 / 今日最佳 / 历史最佳 / 最近一次**
- 贪吃蛇、数独、像素鸟补上难度档位与更清晰的局内控制提示
- 自动化测试继续扩展，补到蓝图恢复、小游戏、输入框编辑等关键回归链路

## 修复与优化

- 持续修复 **蓝图最终输出在重开画布后掉线、跳变、位置回退** 等恢复问题
- 收口蓝图输出拓扑与 placeholder 逻辑，减少 direct pipeline 中间结果误弹出到蓝图外的问题
- 输入 / 输出占位样式与多行提示继续统一，减少遮挡、重复提示和边界异常增长
- 修复功能节点 **Chat / RAG / 自定义 Prompt / 参数输入框** 无法用鼠标拖拽选中文本的问题，并同步补齐任务节点、实验记录、重命名输入框保护
- 小游戏面板改为按当前游戏分别适配高度，避免不同游戏共用固定高度导致内容重叠
- 修复小游戏“按键开始”提示消失后引发的布局跳动，并重新拉开贪吃蛇、像素鸟的速度档位差异
- 收口 AIHubMix / 自定义 OpenAI 兼容 provider 的部分报错提示与模型参数兼容行为
- Copilot 默认模型改为 **`gpt-4.1`**，并移除含混的“自动”选项

## 注意事项

- 这是 **alpha 测试版**，建议先备份工作区，再在副本中试用
