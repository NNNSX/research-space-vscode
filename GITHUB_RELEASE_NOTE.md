这是一次面向多节点画布流畅度、全局缩放视图和画板定位体验的 alpha 更新。

## 新增

- 画布新增 compact / minimal 两级摘要显示，在缩小到全局视角时自动降低节点细节渲染。
- 数据节点和功能节点在摘要态显示轻量标题、类型、状态和核心信息，避免低缩放时内容直接消失。
- 画板在摘要态显示居中大标题导航标签，便于快速定位不同功能区。

## 修复与优化

- 优化几十个节点同屏时的拖拽、选择和缩放流畅度，减少全画布重复重渲染。
- 修复大范围缩放时被旧视口状态拉回原视角的问题。
- 缩小时暂停 PDF 首页、图片、音频波形、Markdown、表格和功能节点参数区等富内容渲染，放大后恢复完整内容。
- 摘要态改为离散固定排版，避免连续字号缩放带来的重排卡顿。
- LOD 切换阈值后移，避免过早进入摘要态；摘要态字号同步放大，提高远视角可读性。
- 画板 resize 角点在缩放过程中保持接近恒定的屏幕尺寸，低缩放下仍可捕捉和拖拽。
- 画板居中大标题已改为独立上层浮层，显示在节点之上且不拦截鼠标事件。

## 注意事项

- 这是 alpha 测试版，建议先备份工作区，再在副本中试用。
- 大量节点同屏时仍会受机器性能、Webview 渲染负载和节点内容类型影响；本版本优先优化全局视角和常见交互掉帧。
