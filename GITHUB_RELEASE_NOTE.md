这是一次面向画布结构化思维组织与导图节点体验的 alpha 更新。

## 新增

- 画布工具栏新增“导图”入口，可创建轻量思维导图节点。
- 思维导图保存为独立 `.rs-mindmap.json` 文件，画布只保存文件路径和轻量摘要。
- 导图编辑器支持大纲编辑与脑图式图形编辑，可维护中心主题、多级分支、折叠展开和图片引用。
- 支持 `.xmind` 拖入 / 导入自动转换为导图节点，并支持导出 `.xmind`。
- 支持导出 Markdown；画布选中导图节点导出 Markdown 时会读取导图文件并导出为大纲。
- 导图节点可连接到文本类 AI 工具，作为 Markdown 大纲内容参与后续整理、摘要和写作。

## 修复与优化

- 脑图图形模式使用统一 SVG 曲线连线层，连接中心主题、分支和子分支。
- 修复大图片分支附近连接线异常包围、重复条目 ID 导致重开后部分分支无连线等问题。
- 带图片分支采用标题胶囊 + 图片卡片的分层排版，图片尺寸可调整，并会同步到大纲与导出结果。
- 画布上的导图节点改为 SVG 缩略图封面，只展示中心主题和一级分支；超过 4 条一级分支时显示“+N 更多”。
- 修复画板编辑弹窗崩溃，以及选中节点悬浮工具条遮挡编辑弹窗的问题。
- 补充导图数据、Markdown 转换、大纲解析、图形编辑和保存重载相关测试。

## 注意事项

- 建议在试用 alpha 版本前备份工作区。
- 当前导图定位为轻量结构化节点，不提供复杂主题系统、多人协作或全画布思维导图布局。
- API Key 只应配置在本机 VS Code 设置中，不应写入项目文件。

## 下载说明

- 下载附件中的 `research-space-2.3.1-alpha.1.vsix`，在 VS Code 中执行 `Extensions: Install from VSIX...` 安装。
